Ce plugin permet de sélectionner des produits à mettre en avant (dans une box de présentation par exemple).

Veuillez simplement glisser le répertoire enavant dans le dossier client/plugins de votre Thelia.


Cette boucle est similaire à la boucle produit:
http://wiki.thelia.net/index.php/La_boucle_PRODUIT

Exemple d'utilisation: 
<THELIA_PRODUIT type="Enavant" >
                  #REF  #TITRE #REWRITEURL    
</THELIA_PRODUIT>
